# Ejercicio 10 - TP2 - Taller de Lenguajes

Este proyecto contiene la resolución del Ejercicio 10, incluyendo las tablas de posiciones por ronda y la tabla final.

## Estructura del Proyecto

- `src/`: Contiene los módulos y funciones de Python necesarios para la resolución.
- `notebooks/`: Contiene el archivo `Ejercicio10.ipynb` para ejecutar la solución de forma interactiva.
- `requirements.txt`: Lista de librerías necesarias.

## Instalación y Configuración

### 1. Preparar el entorno virtual

Desde la terminal, en la raíz del proyecto, ejecuta:

`python3 -m venv .venv`

### 2. Activar el entorno virtual

- Windows

`. .venv\Scripts\Activate`

- Linux

`. .venv\bin\activate`

### 3. Instalar dependencias

Una vez activado el entorno, instala las librerías necesarias:

`pip install -r requirements.txt`

## Ejecución

Para visualizar y ejecutar el ejercicio, inicia Jupyter Lab:

`jupyter lab`

Luego, abre el archivo `Ejercicio10.ipynb` dentro de la carpeta `notebooks/`.
